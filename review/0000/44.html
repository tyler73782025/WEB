單元 15：函數與其圖形 小魔流 topmath 頁碼 1501 1. 變數與常數：在一個關係式中，可變動的數稱為變數，而維持不變的數，稱為常數。 在 y＝3x 中，當 x＝1 時，y＝3，當 x＝2 時，y＝3，因此 x、y 均稱為變數，而 3 為常數。 2. 自變數與應變數：在上例 y＝3x 中，變數 x 的值確定後，變數 y 的值也就隨之確定，因此可以把變數 x 稱為自變數，而變數 y 稱為應變數。 3. 函數：在一個 x 與 y 的關係式中，對於任意一個 x 的值，都恰好可得到一個 y 值與他對應，我們就說 y 是 x 的函數，常用符號 y＝f(x)或 y＝g(x)來表示。 在 y＝3x 中，y 是 x 的函數，可以寫成 y＝f(x)＝3x。 當從 x 到 y 的對應關係是"一對一"或"多對一"時，就稱 y 是 x 的函數。 當從 x 到 y 的對應關係是"一對多"或"一對無"時，則 y 不是 x 的函數。 是函數 不是函數 對應關係 一對一 多對一 一對多 一對無 例子 y＝2x + 1 y＝x 2 ＋1 圖示 1 2 3 5 3 7 -1 0 1 1 2 0 1 1 2 3 0 1 2 1 2 圖示 圖示 x y (1，3) (2，5) (3，7) x y (-1，2) (0，1) (1，2) x y (0，1) (0，2) (1，3) 4. 函數值：若 y 是 x 的函數，當 x＝a 時，所對應的 y 值為 b，就稱 b 為此函數在 x＝a 時的函數值。 函數 y＝3x 中，當 x＝1 時，函數值為 y＝3×1＝3。當 x＝5 時，函數值為 y＝3×5＝15 1. 函數的圖形：若 y 是 x 的函數，則把每個 x 及其對應的 y 值寫成數對(x，y)，並在坐標平面上畫出對 應的點，即可得其函數圖形。 x 0 1 2 y 1 2 x 0 0 1 y 1 2 3 x -1 0 1 y 2 1 2 x 1 2 3 y 3 5 7 單元 Chapter 15 函數與其圖形 主題 1 變數與函數 例 例 註 注意 例 主題 2 函數圖形與線型函數
